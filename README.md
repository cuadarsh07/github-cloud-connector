# ☁️ GitHub Cloud Connector

A high-performance backend REST API built with Python and FastAPI that securely connects to GitHub. This connector demonstrates third-party API integration, secure authentication, and clean code architecture.

## 🚀 Features
* **OAuth 2.0 Authentication (Bonus Feature):** Secure login flow using GitHub's native OAuth app integration.
* **API Integration:** Connects to GitHub's REST API to fetch a user's repositories.
* **Modular Codebase:** Separation of concerns between API routes and external service logic.
* **Modern Tech Stack:** Built with Python, FastAPI, and `httpx` for asynchronous requests.
